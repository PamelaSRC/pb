## Estructura con `char`

```cpp
char c;

cin >> c;

switch (c)
{  
  case 'A':
    cout << "Elegiste A" << endl;
    break;
  case 'B':
    cout << "Elegiste B" << endl;
    cout << "y le sigue a C" << endl;
  case 'C':
    cout << "Elegiste C" << endl;
    break;
  default:
    cout << "Equivalente al else" << endl;
    break;
}  
```

---

## Estructura con `int`

```cpp
int c;

cin >> c;

switch (c)
{  
  case 1:
    cout << "Elegiste 1" << endl;
    break;
  case 2:
    cout << "Elegiste 2" << endl;
    cout << "y le sigue 3" << endl;
  case 3:
    cout << "Elegiste 3" << endl;
    break;
  default:
    cout << "Equivalente al else" << endl;
    break;
}  
```

# Operador `?`

```cpp
int a = 1;
int b = 2;
int c = 0;

if (a == b){
  c = a + b;
}

// Es lo mismo que:
c = (a == b) ? (a + b) : c;
```
